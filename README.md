# vue-todo-list
EN este repositorio vamos a ir desarrollando la aplicación de lista de tareas en vue en diferentes ramas:
- 1-directivas: la aplicación propuesta en el tema de directivas, con un único componente sin usar SFC
- 2-sfc: usando sfc pero toda la aplicación en el App.vue
- 3-componentes: una vez separada la aplicación en componentes, pero no funcionan porque no se ha establecido la comunicación entre ellos
- 4-comunicacion: la aplicación ya funcional usando un store pattern que nos hacemos nosotros
- 5-axios: una vez que añadimos axios para dotarla de persistencia
- 6-router: con diferentes vistas
- 7-pinia: cambiando nuestro store pattern por pinia
